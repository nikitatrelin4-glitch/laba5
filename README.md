# Лабораторная работа: Обход бинарного дерева

## Описание проекта
Данный проект представляет собой консольное приложение на языке C++, демонстрирующее работу со структурами данных типа «Дерево». Программа строит произвольное бинарное дерево, выполняет классические алгоритмы обхода (в глубину и в ширину) и решает задачу подсчета терминальных путей.

* Базовая задача:  Обход бинарного дерева
* Вариативная часть: Задание №4 — Посчитать количество путей от корня до листьев.

## Архитектура проекта
Исходный код структурирован и разделен на логические модули:
* BinaryTree.h — определение структуры узла дерева (Node) и декларация функций обхода и анализа.
* BinaryTree.cpp — реализация механизмов глубокого (рекурсивного) и широкополосного (итеративного с использованием очереди std::queue) обходов.
* tests.h — автоматические модульные тесты на базе макроса <cassert> для проверки корректности обходов на эталонных деревьях.
* main.cpp — демонстрация работы алгоритмов, вывод структуры уровней дерева в консоль.

## Реализованный функционал

### Обязательная часть
1. Прямой обход (Preorder): Корень -> Левое поддерево -> Правое поддерево.
2. Симметричный обход (Inorder): Левое поддерево -> Корень -> Правое поддерево.
3. Обратный обход (Postorder): Левое поддерево -> Правое поддерево -> Корень.
4. Обход в ширину (Level-order / BFS): Поуровневый обход узлов дерева с сохранением структуры уровней.

### Вариативная часть 
* Подсчет путей до листьев: Функция рекурсивно определяет число всех возможных уникальных путей, начинающихся в корневом узле и заканчивающихся в листьях (узлах, не имеющих потомков).
